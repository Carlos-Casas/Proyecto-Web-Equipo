# Proyecto-Web-Equipo
Este proyecto consiste en el desarrollo de un Foro de Deportes, una página web estática donde los usuarios pueden visualizar y publicar comentarios relacionados con diferentes disciplinas deportivas. El objetivo principal es simular el funcionamiento básico de un foro interactivo utilizando tecnologías web fundamentales.

El proyecto fue desarrollado en equipo, organizando el trabajo mediante el uso de ramas en GitHub, donde cada integrante se encargó de una parte específica:

HTML: Estructura y contenido principal de la página.

CSS: Diseño, estilos y apariencia visual del sitio.

JavaScript: Funcionalidad e interacción, como la publicación dinámica de comentarios.

Se utilizó Git y GitHub para el control de versiones, permitiendo trabajar de manera colaborativa, crear ramas independientes para cada área (HTML, CSS y JS) y posteriormente integrar todos los cambios en la rama principal del proyecto.

Este proyecto nos permitió reforzar conocimientos sobre:

Estructura básica de una página web.

Diseño y estilos con CSS.

Manipulación del DOM con JavaScript.

Trabajo colaborativo con control de versiones.
